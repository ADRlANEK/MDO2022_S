1. sklonowanie repozytorium z serwera na komputer lokalny 
	git clone https://github.com/InzynieriaOprogramowaniaAGH/MDO2022_S
	
	
2. utworzenie lokalnego branch'a o nazwie AF400832
	git checkout -b AF400832

3. utworzenie folderu AF400832/Lab01 i pliku sprawozdanie.txt w folderze AF303741/Lab01/sprawozdanie.txt
	mkdir AF400832
	cd AF400832
	touch sprawozdanie.txt

4. git status
	sprawdzenie statusu modyfikowanych/utworzonych/usuniętych plików
	
5. git add .
	dodanie modyfikowanych plików do śledzenia
	
6. git commit -m "[AF303741] Init commit"
	utworzenie commita z śledzonych plików
