Dockerfile
- FROM ubuntu:latest - użycie w obrazie istniejącego już obrazu ubuntu w nanjoweszej wersji
- RUN apt-get update - aktualizacja systemu i jego zależności
- RUN apt install -y git - zainstalowanie gita w systemie

- docker build -t ubuntu-image .
uruchomienie polecenia dockera do stworzenia obrazu przy użyciu Dockerfile znajdującego się w aktualnym katalogu

- docker run -it ubuntu-image
Uruchomienie kontenera z obrazu
