docker login  - zalogowanie się do dockerhub w celu importu repozytorium

docker build - budowanie kontenera z obrazu

docker images - wyświetla kontenery na maszynie

docker tag docker-lab-3:latest yivess/first_docker - dodanie taga do kontenera (wymagany do importu na repozytorium)

docker push yivess/first_docker - push na repo dockerhuba

docker-compose build - zbudowanie kontenera z wykorzystaniem docker-compose.yml

docker-compose up - uruchomienie docker-compose
